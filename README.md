## MIPS-CPU_LOGISIM_NHY

**2017届华科组原课设**

***

- [x] 单周期cpu **24 + 3 + 4**条指令
- [x] 理想流水线CPU
- [x] 气泡流水线CPU
- [x] 重定向流水线CPU
- [x] 单周期MIPS_CPU中断 + 重定向流水线CPU中断

***

**注意**

> 单周期CPU & 理想流水线 —— 寄存器组上升沿，其他——寄存器组下降沿
